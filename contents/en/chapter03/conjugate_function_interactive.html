Understanding the Conjugate Function

Definition: For function f(x), the conjugate is:

$$f^*(y) = \sup_x \{yx - f(x)\}$$

Current Function Analysis:

Key Insights:

  • Geometric Meaning: f*(y) measures the maximum "gap" between the line yx and f(x)
  • Convexity: f* is always convex, even if f is not
  • Fenchel's Inequality: f(x) + f*(y) ≥ xy for all x, y
  • Duality: If f is closed and convex, then f** = f
